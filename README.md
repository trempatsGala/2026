# Trempats · Cena de Gala

Aplicació web per gestionar les comandes de trempats en una cena de gala. Dissenyada per funcionar des del mòbil, sense instal·lació ni servidor.

## Funcionalitats

- Selecció d'ingredients per muntar el trempat personalitzat
- Gestió de comandes per taula
- Interfície elegant adaptada a l'estètica de la gala
- Funciona completament al navegador (HTML + CSS + JS, sense backend)

## Ús

Obre el fitxer `trempats-app.html` directament al navegador, o allotja'l a qualsevol servidor web estàtic (GitHub Pages, Netlify, etc.).

## Tecnologies

- HTML5 / CSS3 / JavaScript (Vanilla)
- Google Fonts (Cormorant Garamond + Montserrat)
